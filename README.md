# 💸 Daily Expenses Tracker

A premium, modern Flutter application designed for seamless financial management. Built with a focus on high-end aesthetics (Glassmorphism), smooth user experience, and robust security.

![Banner](https://images.unsplash.com/photo-1554224155-6726b3ff858f?auto=format&fit=crop&q=80&w=1200)

## ✨ Features

### 🔐 Security & Authentication
- **Secure Access**: Fully functional Login and Registration system.
- **Enhanced Privacy**: Password visibility toggles on all entry forms.
- **Smart Login**: "Remember Me" functionality and secure password reset simulation.
- **Password Management**: In-app password change with real-time validation.

### 👤 Profile Management
- **Identity Control**: Update your full name and email address instantly.
- **Profile Avatars**: Upload custom profile pictures from your gallery or use the auto-generated initial avatars.
- **State Synchronization**: Your profile updates reflect immediately across the entire dashboard.

### 📊 Financial Insights
- **Dynamic Dashboard**: Real-time overview of total balance, income, and expenses.
- **Visual Analytics**: Interactive spend-category charts using `fl_chart`.
- **Export Capabilities**: Generate professional PDF expense reports with one click.

### 🎨 Premium UI/UX
- **Glassmorphic Design**: Modern frosted-glass containers with vibrant background gradients.
- **Responsive Layout**: Optimized for various Android screen sizes.
- **Micro-animations**: Smooth fade-in transitions and interactive hover effects.

## 🚀 Tech Stack

- **Core**: [Flutter](https://flutter.dev) (Dart)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Backend**: [Supabase](https://supabase.com)
- **UI Components**: 
  - `google_fonts` (Typography)
  - `fl_chart` (Data Visualization)
  - `image_picker` (Media Access)
  - `pdf` & `printing` (Document Generation)

## 🛠️ Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Android Studio / VS Code
- Supabase Project URL & Anon Key

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ahmedOsman64/Daily-Expenses.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```
